 Input        | Expected Output |
| ------------ | --------------- |

| `"42"`       | `42`            |

| `"+42"`      | `42`            |
| `"00123"`    | `123`           |
| `"   789"`   | `789`           |

| Input        | Expected Output | Notes                                       |
| ------------ | --------------- | ------------------------------------------- |
| `"abc"`      | `0`             | No digits at all                            |
| `"12abc"`    | `12`            | Stops parsing at first non-digit            |
| `"-12abc"`   | `-12`           | Parses negative, stops at non-digit         |
| `"  +56 78"` | `56`            | Only parses up to the first space after int |
| `"--123"`    | `0`             | Invalid double sign                         |
| `"++123"`    | `0`             | Invalid double sign                         |
| `"+-123"`    | `0`             | Invalid mixed sign                          |

| Input           | Expected Output       | Notes                             |
| --------------- | --------------------- | --------------------------------- |
| `""`            | `0`                   | Empty string                      |
| `"   "`         | `0`                   | Only whitespace                   |
| `"2147483647"`  | `2147483647`          | INT\_MAX for 32-bit               |
| `"-2147483648"` | `-2147483648`         | INT\_MIN for 32-bit               |
| `"2147483648"`  | Undefined / Overflow  | Should detect or handle overflow  |
| `"-2147483649"` | Undefined / Underflow | Should detect or handle underflow |
